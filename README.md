# performance_test
test where we are asked to create a student management system

## what is it for
This code was created to help manage a school
The code shows how each option that the user can choose is managed.
## opcions
A summary of each option:
The first option will ask you to register the student's data (ID, name, age, program, and whether it is active or not). 
If an incorrect option is selected, the user will be told that the option is invalid.

The second option allows us to view the previously entered data.

The third option will ask us to enter the student's name. If the result is satisfactory, it will display all their data. 
Otherwise, it will simply say that the student does not exist.

The fourth option allows us to update data in the previously created list. 
If the user wants to leave things as they are, they should simply press ENTER.

The fifth option allows us to delete students from the previous list and will ask us to confirm.

And the last option simply asks us to exit the program.
